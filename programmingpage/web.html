Webläsare

När man besöker en websida så tolkar webläsaren koden som beskriver websidan. Standarden för webspråk är levande och ändras hela tiden, detta leder till att dom olika webläsarna kapplöper mot utvecklingen av dom olika webspråken.

HTML

HTML är ett språk som beskriver den semantiska strukturen av innehållet i en websida. Dom olika elementen i HTML kallas för taggar och omsluter andra taggar eller text. Innebörden av dom olika taggarna är endast semantisk information, exempelvis markerar man en titel med H-taggen och paragrafer med P-taggen. HTML ska ej användas för att beskriva stil eller utseende. Till detta har vi CSS.

Dagens webläsare är väldigt robusta när det gäller att tolka HTML och med de få taggarna som finns så är kan webläsarna oftast rendera websidornas HTML struktur även om det finns fel i den. Det är dock inte bara webläsare som besöker websidor.

Det finns flera program, kallade robotar eller spindlar, som besöker websidor i olika syften. Exempel på detta är googles robotar som besöker websidor för att indexera dessa. Dessa program renderar inte websidan i en webläsare då detta skulle ta väldigt lång tid. Däremot så försöker google förstå websidans innehåll och då är den semantiska struturen väldigt viktig eftersom att titlar, rubriker och taggar som säger vad som är menyer hjälper robotarna att förstå innehållet.

Yttligare en typ av program som är beroende av korrekt semantik är textläsare som används av folk med synproblem. Att tydligt markera var menyn är semantiskt gör att dessa personer kan hitta menyn utan att behöva låta programmet läsa upp allt innehåll i webläsaren.

CSS

CSS används för att beskriva utseendet på websidan. Det fungerar genom att man specificerar olika egenskaper för olika selektorer. När en webläsare tolkar CSS så läser den in selektorn och tolkar vad den försöker markera för typ av element.

Exempel på en simpel selektor skulle vara selektorn p som markerar alla motsvarande HTML paragraf taggar. Det går även skriva mer komplexa selektorer som exempelvis kan beskriva alla element i tredje kollumnen i en tabell eller alla rubriker som följer ett list-element.

När en webläsare har identifierat vilket eller vilka element selektorn markerar så tolkar den egenskaperna som har definerats. Dessa egenskaper beskriver i regel stil och utseende för det markerade elementet och det kan vara saker som exempelvis teckenstorlek, typsnittsvikt, bakgrundsfärg eller storlek.

CSS används också för att positionera HTML elementen och kan utföra simplare animationer.

Javascript

Javascript är ett prototypbaserat scriptspråk som har varit med länge. Normalt används det för att funtionalitet till en websida men idag har Javascript vuxit och blivit så stort och kraftfullt att det går att göra nästan allt med det.

Idag finns det många dialekter och bibliotek av javascript som ger mer funktionalitet eller definerar ramverk med speciella egenskaper som man kan vilja ha.

Exempel

  • Elm

    - Javascript som skrivs som ett funktionellt programmeringsspråk istället för objektorgienterat som javascript normalt sett skrivs som.

  • Node.js

    - En kraftfull och populär variant av javascript som är i första hand tänkt att köras på servers istället för direkt i webläsaren.

  • Playcanvas

    - En fullt fungerande spelmotor med editor som är implementerad i javascript och som är open source.

  • AJAX

    - Bibliotek som gör det möjligt att asyncront hämta data från servern. Dett möjliggör bland annat att man kan ladda in innehåll i endast en sektion av en websida istället för att ladda om hela websidan.

  • React.js

    - Ett kraftfullt ramverk för att skapa responsiva websidor med dynamisk laddning av data. Rekommenderar starkt att endast använda i kombination med JSX.

  • JQuery

    Ett väldigt populärt lättvikts bibliotek för att skriva Javascript. Syntax skiljer sig lite från klassisk javascript och det tillför ett par färdiga funktioner som ej är implementerade i vanlig javascript. Många nybörjare börjar med JQuery då det är lättare att strukturera och förstå dess syntax men det går att blanda Jquery med andra sorters javascript.
